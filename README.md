# **file-mafia-cli**

A free, open-source File Downloader with Command-line Interface; created with JavaScript/NodeJS. 📁🗃️

---

## **Installation**

```bash
npm i -g file-mafia-cli
```

---

## **Usage**

```bash
fmafia <!url> <!filename_with_ext>
```
